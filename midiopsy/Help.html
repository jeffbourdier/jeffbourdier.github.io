MIDIopsy Help

What is MIDIopsy?

MIDIopsy is used to view, play, and edit the binary contents of standard MIDI files. The user is assumed to be familiar with the standard MIDI file format. (For more information, see Online Resources at the bottom of this page.)

The user interface has an "opening screen" and an "editing screen." When the application is launched, the opening screen is shown. When a file is open, it is shown in the editing screen. The binary contents of the file are shown (in hexadecimal format) on the left, and user-friendly comments are shown on the right. The splitter between the two text boxes can be adjusted as desired. Both text boxes are read-only, unless the file is being edited (see Edit a MIDI file, below). To exit the editing screen and return to the opening screen (effectively closing the file), click Done (or press Esc).

Both text boxes are formatted for readability. Chunks are separated by blank lines. Within each chunk, the type and length are on one line, and the rest of the chunk is divided up into lines based on the chunk type. For MTrk (track) chunks, each event is on its own line.



Opening Screen Commands

These commands can be executed from the opening screen.

Open an existing MIDI file

  • Click Open (or press Ctrl+O).
  • A file that has been opened recently can be reopened directly from the opening screen. Under Open Recent, simply click the file path (or press the corresponding number key).

Create a new MIDI file

  1. Click New (or press Ctrl+N).
  2. Specify the Format.
  3. If the format is 1 (one or more simultaneous tracks) or 2 (one or more independent tracks), specify the Number of tracks.
  4. Under Delta-time division, check either Metrical time or Time-code-based time.
    • For metrical time, specify the Ticks per quarter note.
    • For time-code-based time, specify the SMPTE format (frames per second) and Resolution (ticks per frame).
  5. Click OK (or press Enter). The editing screen is shown with a new MThd (header) chunk created as specified.

Editing Screen Commands

These commands can be executed from the editing screen (when a file is open).

Save a MIDI file

  • To use the file's current name, click Save (or press Ctrl+S). (For a new file being saved for the first time, this is equivalent to Save As.)
  • To use a different name, click Save As (or press Ctrl+Shift+S).

Edit a MIDI file

Warning: MIDI file editing is for expert users only. If not done correctly, it could result in data corruption, making the file unusable. Proceed at your own risk!
  1. Click Start Editing (or press F2). The comments text box disappears, and the text box with the binary file contents (the "hex") becomes editable and expands to fill the space. (You edit the hex, not the comments.)
  2. When finished, click Stop Editing (or press F2). MIDIopsy will attempt to validate the hex.
    • If the hex is valid:
      • It becomes read-only again.
      • It is reformatted.
      • The comments text box reappears, repopulated based on the hex.
    • Otherwise, an error message tells you that the hex is invalid, and you are given a chance to correct or revert your changes.

MIDI file playback

When a MIDI file is opened or saved, MIDIopsy also loads it for media playback. (A new file must be saved before it can be played.) If the file has not been saved since it was last edited, it must be saved in order to play the current version (with the most recent edits). A file cannot be played while it is being edited (and vice versa).

Playback starts at the indicated Starting Position (SP), which can be set to any position within the indicated Duration. When a file is loaded for playback, its Starting Position defaults to 00:00:00, and its Duration is determined. During playback, the current Position is updated.

  • To start playing the file, click Play (or press F5). To stop playback, click Stop (or press F5 again).
  • To reset the Starting Position back to 00:00:00, click Reset SP (or press F6). The starting position should flash orange to indicate that it has been reset.
  • To "synchronize" the Starting Position (i.e., set it equal to the current Position), click Sync SP (or press F7). The starting position should flash yellow to indicate that it has been synchronized.
