Welcome to the home of Jeff Bourdier on GitHub. The main purpose of this site is for me to experiment with HTML, CSS, JavaScript, and SVG. Other than this site, I have five active projects on GitHub at the moment, listed below. They are all free, open-source, native (written in C), and portable (tested on Linux and Windows).

StdJB

The "Jeff Bourdier (JB) standard library" -- a collection of functions, macros, etc. that I use frequently throughout my software.

Plunge

A file synchronization utility similar to rsync.

SCWA

A library for building a "self-contained web application" (SCWA).

SCWAKill

A utility that gracefully shuts down a SCWA.

SCWAL

A SCWA that serves SQLite database files.
