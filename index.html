Welcome to the home of Jeff Bourdier on GitHub. The main purpose of this site is for me to experiment with HTML, CSS, JavaScript, and SVG. Other than this site, I currently have two projects on GitHub, described below. Of course, they are all free, open-source software (FOSS).

MIDIopsy

A WPF application (written in C#) that is used to view, play, and edit the binary contents of standard MIDI files.

MIDIopsy Home Page

Portiveware Suite

A suite of portable, native software ("portiveware") applications, written in C, supported on Windows and Linux, consisting of the following components:

  • Plunge - A command-line utility for synchronizing sets of files.
  • Putter - A web service for handling HTTP PUT requests.
  • Jeff Bourdier (JB) library - Contains types, macros, and functions used by both of the above.

Although developed mainly for my own use, portiveware does offer the following advantages over other applications:

  • Speed
  • Simplicity
  • Portability
  • Small footprint
  • No third-party dependencies
